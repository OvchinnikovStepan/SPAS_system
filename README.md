# SPAS System 

## Запуск проекта

**Порты:** `8000`, `8501`  
**URL:** `http://localhost:8501`


## Подготовка

Перед запуском необходимо отформатировать названия некоторых файлов, а именно: удалить часть 
.example из имени файла и названия папки .env.example, находящегося в папке /api/, .streamlit.example, 
находящейся в папке /dashboard/app/

## Docker
**Для запуска в фоновом режиме, в корневой папке прописать:**
```bash
docker-compose up -d
```

**Для остановки, там же прописать:**
```bash
docker-compose down -v
``` 

**Для запуска в консольном режиме, в корневой папке прописать:**
```bash
docker-compose up --build
```
**Для остановки нажать:** 
Ctrl+C + Ctrl+S
