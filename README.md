# SPAS System 

## Запуск проекта

**Порты:** `8000`, `8501`  
**URL:** `http://localhost:8501`


## Подготовка

Перед запуском удалите из названия файла .env.example в папке api/ и папки .streamlit.example папке dashboard/app 
часть .example

## Docker
**Для запуска в фоновом режиме, в корневой папке прописать:**
```bash
docker-compose up -d
```

**Для остановки, там же прописать:**
```bash
docker-compose down -v
``` 

**Для запуска в консольном режиме, в корневой папке прописать:**
```bash
docker-compose up --build
```
**Для остановки нажать:** 
Ctrl+C + Ctrl+S
